# DuneTripSafari - Full-Stack Tourism Booking Web App

## 🌟 Overview
**DuneTripSafari** is a fully functional tourism booking platform built with **Django, Django REST Framework, React, and PostgreSQL**.  
It is **live, actively used**, and demonstrates **real-world full-stack development, bookings, notifications, and scalable architecture**.

**Live Site:** [https://www.dunetripsafari.com](https://www.dunetripsafari.com)

---

## 🛒 Key Features

**User / Tourist:**
- Browse available tours and packages
- Add tours to cart and checkout
- View personal bookings on the website
- Receive email confirmations for bookings
- Real-time notifications and booking updates

**Admin / Owner:**
- Manage tour packages and availability
- Track bookings and payments
- Send automated emails to customers


**Marketing & Analytics:**
- Meta Pixel integration for conversion tracking
- Understanding user behavior and bookings
- Supports ad campaigns with conversion API

**System Highlights:**
- Full-stack architecture with **React frontend + Django backend**
- **PostgreSQL** database for booking and user management
- Scalable and production-ready deployment
- Dockerized for easier development and deployment

---

## 🛠 Tech Stack
- **Backend:** Python, Django, Django REST Framework  
- **Frontend:** React, JavaScript, HTML, CSS  
- **Database:** PostgreSQL  
- **Deployment:** VPS with Nginx + Gunicorn  
- **Email / Notifications:** Django email system  
- **Analytics & Ads:** Meta Pixel + Conversion API  
- **Containerization:** Docker (frontend, backend, database)
